Ce este ea?

Căutarea binară este un algoritm eficient de căutare, folosit de obicei pentru a căuta o anumită valoare într-un vector sortat, de dimensiuni mari.

Cum funcționează?

Căutarea binară funcționează prin divizarea repetată a listei în jumătate și compararea elementului căutat cu elementul din mijlocul listei. În funcție de rezultatul acestei comparații, căutarea continuă în jumătatea corespunzătoare a listei inițiale. Procesul se repetă până când elementul căutat este găsit sau până când lista se reduce la o dimensiune minimă. Algoritmul de căutare binară folosește tehnica ”divide et impera” sau ”divide și cucerește”. Ideea de bază a acestui algoritm este simplă: după fiecare pas, lista sau vectorul de căutare este împărțit în două părți egale. Această abordare are avantajul că, odată ce lista este ordonată, numărul de comparații necesare pentru a găsi un element este redus semnificativ.

Pentru a înțelege cum funcționează acest algoritm, să parcurgem pașii săi de bază:

1. Inițializarea variabilelor

Înainte de a începe căutarea, avem nevoie de trei variabile:

Left – reprezintă începutul intervalului în care căutăm;

Mid – reține mijlocul intervalului;

Right – indică sfârșitul intervalului de căutare.

2. Verificarea elementului mijlociu

În fiecare pas, algoritmul verifică dacă elementul din mijlocul intervalului (V[Mid]) este egal cu elementul căutat (x). Dacă da, atunci algoritmul a găsit elementul și căutarea se încheie.

3. Actualizarea intervalului

Dacă elementul din mijloc (V[Mid]) nu este cel căutat, intervalul de căutare este actualizat:

Dacă X este mai mic decât V[Mid], atunci Right devine Mid – 1.

Dacă X este mai mare decât V[Mid], atunci Left devine Mid + 1.

4. Repetarea până la găsire

Acești pași se repetă atât timp cât Left <= Right. Dacă acest lucru nu mai este valabil, înseamnă că elementul nu se află în lista dată.
