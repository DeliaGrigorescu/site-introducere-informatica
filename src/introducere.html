Ce este un vector?

Un tablou unidimensional este o serie de elemente ale aceluiași tip de date, ce poartă numele de vector .

Cum se declară în C++?

Elementele sunt indexate de la zero, adică primul element este pe poziția 0 sau are indicele 0. Un vector are nevoie de trei valori pentru a fi declarat: tipul vectorului, numele și dimensiunea, care arată ce tip de date au elementele din vector, cum se numește vectorul și câte elemente poate avea în total.

Exemplu:

int v[7];

Deși are un anumit spațiu rezervat la declarare, el nu trebuie să fie ocupat în întregime, poate fi ocupat parțial. Important este ca numărul de elemente specificat să nu depășească dimensiunea vectorului.

Cum să adaugi elemente în vector?

Există puține metode prin care putem aloca stoca valori într-un vector. Iată două dintre cele mai comune metode:

1. Inițializare directă

int a[7] = {1, 5, 6, 45, 2, 200, 1};

2. Metoda de introducere a utilizatorului

int a[7];

for (int i = 0; i < 7; i++)

cin >> a[i];

Afișarea unui vector

int a[3] = {1, 2, 3};

for (int i = 0; i < 3; i++)

cout << a[i] << ' ';

Schimbarea unei valori

Dacă dorim să schimbăm un anumit element stocat într-o matrice, vom folosi următoarea metodă:

int a[5] = {1, 2, 3, 4, 5};

a[0] = 10;
